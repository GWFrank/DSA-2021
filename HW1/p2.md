```pseudocode
ReverseQueue(source, helper)
    n = source.size()
    for(i=0; i<n; i++)
        for(j=0; j<i-1; j++)
            tmp = source.dequeue()
            source.enqueue(tmp)
        tail = source.dequeue()
        helper.enqueue(tail)
    for(i=0; i<n; i++)
        tmp = helper.dequeue()
        helper.enqueue(tmp)
```

### 2.

Because `enqueue`, `dequeue` and `size` all take $O(1)$ time, the time complexity is

$O(1) + O(n \cdot (n+1)) + O(n) = O(1) + O(n^2) + O(n) = O(n^2)$
